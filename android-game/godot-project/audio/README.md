# Audio Assets

Place your audio files here according to the folder structure:

## Music (OGG Vorbis format)
- `music/menu_theme.ogg` - Main menu background music
- `music/combat_theme.ogg` - Combat/gameplay music
- `music/boss_theme.ogg` - Boss battle music
- `music/victory.ogg` - Victory fanfare

## Sound Effects (OGG Vorbis format)
- `sfx/` - All game sound effects
- See `AudioManager.gd` for complete list

## Voice Over (OGG Vorbis format)
- `vo/` - Shaman wisdom and voice lines

## Creating Audio
See `ASSET_CREATION_GUIDE.md` in the root directory for detailed instructions on creating and exporting audio files.
